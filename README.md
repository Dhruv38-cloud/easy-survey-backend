* Clone the repository
  - git clone https://github.com/Dhruv38-cloud/easy-survey-backend.git

* Install dependencies
  - bundle install

* Database creation
  - rails db:create

* Database initialization
  - rails db:migrate

* Run server
  - rails s -p 8080
